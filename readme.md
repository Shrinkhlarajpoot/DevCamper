# DEVCAMPER API

> Backend Api for devcamper api which is a bootcamp website

## Usage

You hav to replace config/config.env.env with config/config.env and set the values/settings of your own.

## Install Dependencies
```
npm install
```
## Run app

```
#Run in dev mode
npm run dev

#Run in prod mode
npm  start
```
